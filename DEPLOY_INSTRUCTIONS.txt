   YT Remote — Deploy en Netlify (GRATIS)
═══════════════════════════════════════════════════

OPCIÓN A — Drag & Drop (más fácil, 1 minuto):
──────────────────────────────────────────────
1. Ve a https://app.netlify.com
2. Crea una cuenta gratis (o entra con GitHub/Google)
3. En el dashboard, busca el recuadro que dice
   "Deploy manually" o "drag and drop your site folder here"
4. Arrastra esta carpeta (dist_deploy) directamente ahí
5. ¡Listo! Netlify te da una URL tipo:
   https://random-name-123.netlify.app

OPCIÓN B — Vercel (también gratis):
────────────────────────────────────
1. Ve a https://vercel.com
2. Crea cuenta gratis
3. Instala Vercel CLI: npm i -g vercel
4. Desde esta carpeta ejecuta: vercel --prod
5. Sigue las instrucciones en consola

OPCIÓN C — GitHub Pages:
─────────────────────────
1. Sube los archivos a un repo de GitHub
2. Ve a Settings > Pages
3. Selecciona la rama y carpeta raíz
4. GitHub te da una URL pública
