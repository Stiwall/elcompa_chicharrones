# YT Remote 🎬

Control remoto para YouTube — reproduce y sincroniza videos con amigos en tiempo real.

## Deploy en GitHub Pages

1. Sube el contenido de esta carpeta a tu repositorio
2. Ve a **Settings → Pages**
3. Selecciona `main` branch, carpeta `/root`
4. Guarda — tu app estará en `https://tu-usuario.github.io/nombre-repo`

## Tecnologías
- React 19 + TypeScript
- Tailwind CSS + shadcn/ui  
- PeerJS (conexión P2P)
- YouTube IFrame API
